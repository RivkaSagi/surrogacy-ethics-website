

אתר סטטי מבוסס [Next.js 15](https://nextjs.org/) + Tailwind CSS שמושך בזמן אמת את התוכן משלושה Google Docs וגיליון Google Sheet:

- `מי אנחנו` – מסמך Google Docs
- `תמצית הקוד האתי` – מסמך Google Docs קצר
- `הקוד האתי` – מסמך Google Docs
- `רשימת החותמות והחותמים` – Google Sheet (CSV)
- `עדכונים` – מסמך Google Docs לתיבת הגלילה

הנתונים מנוהלים בקובץ ההגדרות `src/config/content.ts`. ניתן להחליף שם מזהי מסמכים/גיליון וכתובת אימייל ליצירת קשר.

## התקנה והרצה

```bash
npm install
npm run dev    # http://localhost:3000
npm run lint   # בדיקות ESLint
npm run build  # הפקת גרסת פרודקשן
```

## פרסום המסמכים לציבור

כדי שהמשתמשים יראו את התוכן ללא התחברות חובה לפרסם כל מסמך/גיליון:

1. בתוך Google Docs/Sheets בחרו `File → Share → Publish to web`.
2. ודאו שהקישור ציבורי (`Anyone with the link`).
3. העתיקו את מזהה המסמך (החלק שבין `/d/` ל-`/edit`) והכניסו ל-`src/config/content.ts`.

> ללא פרסום, Google תחסום את בקשת ה-`fetch` מהדפדפן עקב חוסר הרשאות.

### מבנה הגיליון

הטבלה קוראת את העמודות הראשונות בלבד:

| עמודה | תוכן            |
| ----- | ---------------- |
| A     | שם החותם/ת      |
| B     | תחום מקצועי/טקסט|

שורות ריקות יופעלו אוטומטית.

## התאמות עיצוב ותוכן

- **לוגו** – כרגע מוצג עיגול עם האות ל׳. החליפו ב-`SiteHeader` או בשדה הרקע בקובץ `hero-section`.
- **שפה וכיוון** – `layout.tsx` מגדיר `lang="he"` ו-`dir="rtl"`.
- **פונט** – משלבים Google Fonts `Assistant` ו-`Secular One`. ניתן להחליף בקלות בקובץ `layout.tsx`.
- **לוגו/תמונה** – הקבצים `public/logo.jpeg` ו-`public/hero.jpeg` מגיעים מהתקיה Downloads; החליפו אותם בחומרים החדשים במידת הצורך.
- **תיבת עדכונים** – נטענת מהדוק ייעודי ונגללת לגובה 320px (`UpdatesPanel`).
- **צור קשר** – הכפתור הוא `mailto:` בלבד. החליפו את `CONTACT.email` לטובת התיבה שלכם.
- **הקוד האתי המלא** – נמצא גם בעמוד ייעודי בכתובת `/code` עבור קריאה נוחה למסמך הארוך.

## פריסה

לפריסה ב-[Vercel](https://vercel.com/) או בספק דומה:

```bash
npm run build
npm start   # הרצה של גרסת הפרודקשן
```

אין צורך בשרת ייעודי – Next.js מרנדר את הנתונים בצד הלקוח (`fetch` בזמן אמת). ודאו שמדיניות ה-CORS של המסמכים נשארת ציבורית גם לאחר פריסה.
